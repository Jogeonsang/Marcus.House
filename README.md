
# marcus-house

### 환경 설정
#### 시스템 요구 사항
- node.js version 12 이상

#### 명령어
```
# 외부 의존성 패키지 설치
$ yarn

# localhost:3000으로 개발환경 빌드 및 실행
$ yarn start:dev

# 프로덕션 환경 실행
$ yarn start:prod

# 개발 환경 빌드
$ yarn build:dev 

# 프로덕션 환경 빌드
$ yarn build:prod
```
